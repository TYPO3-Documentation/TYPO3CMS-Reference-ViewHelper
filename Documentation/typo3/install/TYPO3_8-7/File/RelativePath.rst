=================
file.relativePath
=================


Get file path relative to PATH_site from absolute path

Examples
========

Defaults::

   <f:file.relativePath>/var/www/typo3/instance/typo3temp/foo.jpg</f:file.relativePath>

Output::

   typo3temp/foo.jpg

Arguments
=========


This ViewHelper has no arguments.
