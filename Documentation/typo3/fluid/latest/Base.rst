====
base
====


ViewHelper which creates a :html:`<base href="..."></base>` tag.

The Base URI is taken from the current request.

Examples
========

Example::

   <f:base />

Output::

   <base href="https://example.org/" />

Depending on your domain.

Arguments
=========


This ViewHelper has no arguments.
