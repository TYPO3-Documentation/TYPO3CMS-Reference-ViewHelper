.. _typo3-fluid-case:

====
case
====


Case view helper that is only usable within the SwitchViewHelper.

Arguments
=========


.. _case_value:
value
-----

:aspect:`DataType`
   mixed

:aspect:`Required`
   true
:aspect:`Description`
   The switch value. If it matches, the child will be rendered.

.. _case_default:
default
-------

:aspect:`DataType`
   mixed

:aspect:`Required`
   true
:aspect:`Description`
   If this is set, this child will be rendered, if none else matches.
