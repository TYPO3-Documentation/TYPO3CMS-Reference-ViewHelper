================
renderRenderable
================


Render a renderable.

Set the renderable into the \TYPO3\CMS\Form\Mvc\View\FormView
and return the rendered content.

Scope: frontend

Arguments
=========


.. _renderrenderable_renderable:
renderable
----------

:aspect:`DataType`
   mixed

:aspect:`Required`
   true
:aspect:`Description`
   A RenderableInterface instance
